# User Guide

Welcome to the **Leading University Competitive Programming Archive**. This comprehensive guide will help you navigate and make the most of our platform.

---

## Getting Started

Before you can access all features, you need to create an account:

1. Click the **"Get Started / Login"** button on the home page
2. Sign in using your **GitHub** account
3. That's it! ✨

---

## User Types

There are 3 types of users in LU-CP-Archive:

1. <UserTypeBadge user_type="STANDARD" /> **User** — Default user type for all new users
2. <UserTypeBadge user_type="POWER" /> **User** — Assigned by admins for trusted contributors
3. <UserTypeBadge user_type="ADMIN" /> **User** — Full platform access and moderation capabilities

Your account type determines which features and actions you can access on the platform.

### Standard User & Power User Features

**<UserTypeBadge user_type="STANDARD" /> Users & <UserTypeBadge user_type="POWER" /> Users** can:
- View their contribution statistics
- Connect and manage their Codeforces handle
- Control their leaderboard visibility with the **"Appear on Leaderboard"** toggle

### Admin Features

**<UserTypeBadge user_type="ADMIN" /> Users** can:
- Do everything Standard and Power users can do
- Manage other users' roles:
  - Promote users from Standard to Power
  - Demote users from Power to Standard
  - Step down from admin role to become a Standard user

## Contests

The platform offers three types of competitive contests:

- **Intra LU Contests** — Problems from competitive programming contests organized within LU
- **Short Contests** — Focused, time-limited competitions organized by LU Problem Solvers
- **Marathon Contests** — Extended competitions for comprehensive practice organized by LU Problem Solvers

### Permissions by User Type

**<UserTypeBadge user_type="STANDARD" /> Users** can:
- Submit contests
- Update contest status ("Done", "In Progress", "Skipped")
- Filter contests by difficulty ("Easy", "Medium", "Hard")

**<UserTypeBadge user_type="POWER" /> Users & <UserTypeBadge user_type="ADMIN" /> Users** can:
- Do everything Standard users can do
- Review and approve contests submitted by other users
- Modify or delete existing contests
